Q & A

Web 検索はまだ存在しているのでは?
クエリに関係ない、または関連性の低いものばかりを返すものは検索ではない。
まだ生きているサイトについても Wayback Machine にリンクしている理由は?
サイトが消滅し、その後内容が別物に置き換わる可能性があるため。
